# Activity Overview

## Scenario
You are a level one SOC analyst at a financial services company. You received an alert about a suspicious file downloaded on an employee's computer. The investigation revealed that the file was malicious.

## Step-by-Step Instructions
1. Use the Pyramid of Pain template.
2. Review the alert details.
3. Enter the file hash into VirusTotal.
4. Analyze the VirusTotal report.
5. Determine whether the file is malicious.
6. Fill in the Pyramid of Pain template with additional IoCs.
